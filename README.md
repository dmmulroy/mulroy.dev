# mulroy.dev

Personal shadcn registry hosting reusable TypeScript utilities.

## Registry

### Option

Option type for representing optional values without null/undefined.

```bash
npx shadcn@latest add https://registry.mulroy.dev/r/option.json
```

### Redacted

Branded type preventing sensitive data from logging/serialization.

```bash
npx shadcn@latest add https://registry.mulroy.dev/r/redacted.json
```

### Result

Result type for typed error handling without exceptions.

```bash
npx shadcn@latest add https://registry.mulroy.dev/r/result.json
```

### TaggedError

Mixin factory for discriminated error classes with typed props.

```bash
npx shadcn@latest add https://registry.mulroy.dev/r/tagged-error.json
```
